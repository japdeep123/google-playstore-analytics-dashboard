# 📊 Google Play Store Analytics Dashboard

An interactive, live web dashboard analyzing Google Play Store app data — built as part of a Data Analytics internship task at **Elevance Skills**.

## 🔍 Overview

This project transforms raw Google Play Store data into 6 interactive visualizations, each rendering with its own staggered "live" animation timing to create a dynamic dashboard experience.

## 🛠️ Tech Stack

- **HTML5** — structure
- **CSS3** — styling, layout, animations
- **Plotly.js** — interactive charting library
- **JavaScript (Vanilla)** — data loading & staggered chart rendering logic
- **Python (Jupyter Notebook)** — data cleaning & exploration
